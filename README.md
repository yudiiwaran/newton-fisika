Aplikasi web edukasi interaktif yang dirancang untuk membantu siswa memahami konsep dasar **Hukum Newton** melalui simulasi visual dan kalkulator dinamis. Project ini dibangun menggunakan **React** dengan arsitektur **Atomic Design** untuk modularitas kode yang maksimal.

---

## üåü Fitur Utama

### 1. Materi Interaktif (Lessons)
Penjelasan mendalam mengenai Hukum Newton I, II, dan III lengkap dengan rumus fisika dan contoh penerapan nyata dalam kehidupan sehari-hari.

### 2. Simulasi Hukum I (Inersia)
Visualisasi konsep kelembaman benda menggunakan simulasi koin dan kertas. Pengguna dapat melihat perbedaan hasil antara tarikan lambat dan tarikan cepat.

### 3. Kalkulator & Simulasi Hukum II (F = m √ó a)
- **Kalkulator:** Input massa dan percepatan untuk mendapatkan nilai gaya secara presisi.
- **Simulasi Dorong Kotak:** Visualisasi pergerakan objek berdasarkan gaya (N) dan massa (kg) yang diinput melalui slider.

### 4. Simulasi Hukum III (Aksi-Reaksi)
Simulasi dua objek yang saling berinteraksi. Menampilkan indikator gaya aksi dan reaksi secara *real-time* (N) untuk menunjukkan bahwa gaya selalu berpasangan dan berlawanan arah.
